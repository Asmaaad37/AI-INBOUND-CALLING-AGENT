
# Urdu NLU Module

This module performs:

- Intent Classification
- Named Entity Recognition (NER)

Model:
- XLM-RoBERTa Base

Features:
- Urdu language understanding
- English normalized JSON output
- Multitask learning architecture

Example Output:

{
  "intent": "fee_query",
  "entities": {
    "PROGRAM": "BSCS"
  }
}
